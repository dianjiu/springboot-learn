# springboot-learn  
本项目主要记录自己学习SpringBoot的源码和笔记，项目比较初级，主要目的是用起来，至于原理自行看源码研究。

## 项目说明

开发工具

IDEA 2020.3

开发环境

JDK		13.0.2

SpringBoot		2.4.1
